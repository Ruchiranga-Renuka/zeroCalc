# 🧮 Simple Calculator

A simple, fully functional web calculator with a vibrant dark-themed UI. Built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

## ✨ Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Percentage (`%`) and sign toggle (`+/-`) support
- Decimal point input with duplicate-dot prevention
- Expression history shown above the result
- Division by zero and invalid input protection
- Responsive button press animations
- Colorful gradient buttons on a dark navy background
